# assembler (n)

/əˈsemblə(r)/

## &bull; a [person](../p/person-n.md#•-a-human-as-an-individual) who [assembles](assemble-v.md#•-assemble-something-to-fit-together-all-the-separate-parts-of-something-for-example-a-piece-of-furniture-ráp) a [machine](../m/machine-n.md#•-a-piece-of-equipment-with-many-parts-that-work-together-to-do-a-particular-task) or its [parts](../p/part-n.md#•-part-of-something-some-but-not-all-of-a-thing)

## &bull; (computing) a [program](../p/program-n.md#•-a-set-of-instructions-in-code-that-control-the-operations-or-functions-of-a-computer) for changing [instructions](../i/instruction-n.md#•-a-piece-of-information-that-tells-a-computer-to-perform-a-particular-operation) into [machine](../m/machine-n.md#•-a-computer) [code]()

## &bull; (computing) assembly language, the language in which a [program](../p/program-n.md#•-a-set-of-instructions-in-code-that-control-the-operations-or-functions-of-a-computer) is written before it is changed into [machine](../m/machine-n.md#•-a-computer) code
