# incur (v)

- /ɪnˈkɜː(r)/ [🔊](https://www.oxfordlearnersdictionaries.com/media/english/uk_pron/i/inc/incur/incur__gb_1.mp3)
- /ɪnˈkɜːr/ [🔊](https://www.oxfordlearnersdictionaries.com/media/english/us_pron/i/inc/incur/incur__us_1.mp3)

(formal)

## incur something - if you incur something unpleasant, you are in a situation in which you have to deal with it (phải gánh chịu; phải gánh/chịu/nhận)

- By using an array to implement double-ended queues, we incur the cost of implicitly resizing it.
